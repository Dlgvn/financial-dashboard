## Visual Identity

**Theme: Dark OLED**
- Recommended for financial/fintech dashboards
- Easy on the eyes for long analysis sessions
- High contrast makes numbers pop

**Color Palette:**
| Role | Color | Use |
|------|-------|-----|
| Background | `#020617` (near black) | Page base |
| Surface | `#0F172A` (navy dark) | Cards, panels |
| Elevated | `#1E293B` (slate dark) | Hover states, modals |
| Accent | `#22C55E` (green) | Positive metrics, CTAs, healthy scores |
| Danger | `#EF4444` (red) | Negative metrics, red flags, distress |
| Caution | `#F59E0B` (amber) | Warning states, grey zone |
| Text | `#F8FAFC` (near white) | Primary text |
| Muted | `#94A3B8` (slate 400) | Labels, secondary text |

**Typography: IBM Plex Sans**
- Financial, trustworthy, professional
- Used by Bloomberg terminals and major fintech products
- All weights: 300 / 400 / 500 / 600 / 700
- Google Fonts: https://fonts.google.com/share?selection.family=IBM+Plex+Sans:wght@300;400;500;600;700

## Key UX Principles

| Principle | Application |
|-----------|-------------|
| **Numbers first** | Large, prominent metric displays — not buried in tables |
| **Color = meaning** | Green always = good, Red always = bad, Amber = caution. Consistent everywhere. |
| **Progressive disclosure** | Retail users see score + summary. Analysts click to expand raw ratios. |
| **Mongolian context** | Language toggle (MN/EN) in top bar. MNT inflation toggle persistent. |
| **Breadcrumbs** | Always show: `Screener > APU > Forensic Analysis` |
| **Empty states** | "Upload your first company to get started" — never a blank screen |

## Accessibility & Quality Checklist

- [ ] No emojis as icons — use SVG (Lucide / Heroicons)
- [ ] All clickable elements have `cursor-pointer`
- [ ] Hover states with smooth transitions (150–300ms)
- [ ] Minimum 4.5:1 contrast ratio for all text
- [ ] Focus states visible for keyboard navigation
- [ ] `prefers-reduced-motion` respected
- [ ] Responsive at 375px, 768px, 1024px, 1440px
- [ ] Tables wrapped in `overflow-x-auto` for mobile
- [ ] Color is never the only indicator (always paired with label/icon)
