**2. Create `financial_dashboard/scraper/price_scraper.py`** (per D-08, D-09, D-11):

Must contain these exports:
- `scrape_company_prices(mse_id: int, company_name: str) -> list[dict]` — Fetches `http://old.mse.mn/en/company/{mse_id}`, parses the `<table class="trade_history_result">` from `<tbody>`, extracts cells in order `[No, High, Low, Open, Close, Volume, Value, Date]`. Strips commas from numeric fields. Returns list of dicts with keys: `date`, `open`, `high`, `low`, `close`, `volume`. Sorted by `date` ascending. Raises `ValueError` if no `trade_history_result` table found. Uses `requests.get()` with `headers={"User-Agent": "Mozilla/5.0 (compatible; MSEAnalytica-Scraper/1.0)"}` and `timeout=30`.
- `save_price_data(company_name: str, mse_id: int, records: list[dict]) -> str` — Creates `PRICES_DIR / price_filename(company_name)`. JSON structure: `{"company": company_name, "mse_id": mse_id, "scraped_at": datetime.now().isoformat(), "records": sorted_records}`. Uses `ensure_ascii=False, indent=2`. Creates `data/prices/` dir if absent. Returns filename.
- `price_file_exists(company_name: str) -> bool` — Checks if `PRICES_DIR / price_filename(company_name)` exists.
- `price_filename(company_name: str) -> str` — Sanitizes name: `name.strip().strip('"').strip()`, removes filesystem-unsafe chars (`"`, `/`, `\`, `:`, `*`, `?`, `<`, `>`, `|`), returns `f"{name}.json"`.

Constants:
```python
DATA_DIR = Path(__file__).parent.parent.parent / "data"
```
