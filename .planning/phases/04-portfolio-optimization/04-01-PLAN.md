**Constants:**
```python
PRICES_DIR = Path(__file__).parent.parent.parent / "data" / "prices"
SECTOR_COLORS = {"Banking": "#60a5fa", "Insurance": "#f59e0b", "Standard": "#4ade80"}
```

**8 exported functions:**

1. `load_price_returns(company_names: list[str]) -> dict[str, np.ndarray]`
   - For each name, read `PRICES_DIR / f"{name}.json"`, parse records, extract `float(r["close"])` values
   - Compute log returns: `np.log(closes[1:] / closes[:-1])`
   - Skip companies with missing files or <2 records
   - Return `{company_name: log_return_array}`

2. `align_returns(returns_map: dict[str, np.ndarray]) -> tuple[list[str], np.ndarray]`
   - Find min length across all return arrays
   - Truncate each to last `min_len` observations (most recent)
   - Return `(names_list, np.column_stack matrix [n_obs x n_assets])`

3. `compute_portfolio_returns(weights: np.ndarray, returns_matrix: np.ndarray) -> np.ndarray`
   - Return `returns_matrix @ weights`

4. `compute_risk_metrics(portfolio_returns: np.ndarray) -> dict`
   - Guard: if len < 10, return `{"sortino": None, "max_drawdown": None, "cvar_95": None}`
   - Sortino: `ann_return / (downside_std * sqrt(252))` where downside = returns < 0, target = 0
   - Max Drawdown: from cumulative product of (1 + returns), find max peak-to-trough
   - CVaR 95%: `np.percentile(returns, 5)` threshold, average of returns <= threshold
   - Return dict with float values or None

5. `mean_variance_optimize(returns_matrix: np.ndarray, names: list[str]) -> dict`
   - Annualize: mean_returns * 252, cov_matrix * 252
   - Objective: negative Sharpe ratio (risk_free=0): `-(w @ mean_returns) / sqrt(w @ cov @ w)`
   - Constraints: `sum(w) == 1`, bounds `(0, 1)` per asset
   - Use `scipy.optimize.minimize(method="SLSQP", options={"maxiter": 1000, "ftol": 1e-9})`
   - Return `{"weights": {name: round(w*100, 1) for each}, "success": bool}`
   - Fallback on failure: equal weights

6. `sample_frontier(returns_matrix: np.ndarray, n_samples: int = 200) -> list[dict]`
   - Use `np.random.default_rng(seed=42)` for reproducibility
   - Generate n_samples random weight vectors, normalize to sum=1
   - For each: compute annualized return (%) and risk (std dev %)
   - Return `[{"risk": str(round(risk, 2)), "return": str(round(ret, 2))}, ...]`
   - ALL values as strings (Reflex list[dict[str,str]] constraint per D-07)

7. `rebalance_weights(holdings: list[dict], company: str, new_pct_str: str) -> list[dict]`
   - Parse new_pct from string, clamp to 0-100
   - Set target company to new_pct
   - Scale all other holdings proportionally so total = 100
   - If other_total == 0, distribute remainder equally among others
   - Update each holding's `weight` (0-1 float), `weight_pct` (string "25.0"), `weight_str` ("25.0%")
   - Return new holdings list (do NOT mutate input)

8. `compute_sector_breakdown(holdings: list[dict]) -> list[dict]`
   - Aggregate `float(h["weight_pct"])` by `h.get("sector", "Standard")`
   - Return `[{"name": sector, "value": str(round(pct, 1)), "fill": SECTOR_COLORS[sector]}, ...]`
   - Only include sectors with pct > 0

All numeric values in returned dicts that feed Recharts MUST be strings: `str(round(value, N))`.
