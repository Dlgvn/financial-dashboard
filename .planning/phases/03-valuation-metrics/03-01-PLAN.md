**Part B: Create financial_dashboard/analysis/valuation.py (per D-01 through D-05)**

Create a new module with a single public function:

```python
def compute_valuation_metrics(
    parsed_data: dict,
    shares_outstanding: int | None,
    last_close_price: float | None,
) -> dict:
    """Compute 4 valuation ratios.

    Args:
        parsed_data: The company's financial JSON (balance_sheet, income_statement, cash_flow).
        shares_outstanding: Total shares outstanding (scraped or manual).
        last_close_price: Most recent close price from price JSON records.

    Returns:
        {
            "ev_ebitda": float | None,
            "fcf_yield": float | None,
            "pe": float | None,
            "pbv": float | None,
            "market_cap": float | None,
            "ev": float | None,
        }
    """
```

Implementation details:

- **Market Cap** = `shares_outstanding * last_close_price`. If either is None, all ratios return None.

- **EV/EBITDA (per D-03):**
  - `total_debt = short_term_loans + long_term_loans` (from balance_sheet; either can be None, treat as 0 if None)
  - `cash = cash_and_equivalents` (from balance_sheet)
  - `EV = market_cap + total_debt - cash` (if cash is None, treat as 0)
  - `ebitda = ebit = profit_before_tax + financial_expense` (same approximation as ratios.py line 117-125; if financial_expense is None, ebitda = profit_before_tax)
  - `ev_ebitda = EV / ebitda` (None if ebitda is None or 0)

- **FCF Yield (per D-02):**
  - `ocf = operating_cash_flow` (from cash_flow)
  - `capex = investing_cash_flow` (from cash_flow — this is negative for outflows)
  - `fcf = ocf - abs(capex)` if both exist. Use `abs(investing_cash_flow)` since investing CF is typically negative.
  - If `investing_cash_flow` is None, fall back to `fcf = ocf`.
  - `fcf_yield = fcf / market_cap` (None if market_cap is 0 or None)

- **P/E (per D-04):**
  - `net_income = income_statement["net_income"]`
  - `pe = market_cap / net_income` (None if net_income is None, 0, or negative)

- **P/BV (per D-04):**
  - `book_value = total_equity` (from balance_sheet; derive as `total_assets - total_liabilities` if missing)
  - `pbv = market_cap / book_value` (None if book_value is None or <= 0)

Use a local `_safe_div(numerator, denominator)` helper that returns None if denominator is None or 0.

Use current-period fields only (no `_prev` suffix).
