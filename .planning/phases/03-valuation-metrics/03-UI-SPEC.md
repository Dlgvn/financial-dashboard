## Spacing Scale

Declared values (must be multiples of 4). Extracted from existing codebase patterns.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (`gap-1`), inline label-to-value spacing |
| sm | 8px | Compact inter-element spacing (`spacing="2"`) |
| md | 16px | Default element spacing (`spacing="4"` = 16px in Reflex) |
| lg | 24px | Section padding (`spacing="6"`) |
| xl | 32px | Layout gaps between card rows |
| 2xl | 48px | Empty/placeholder vertical padding (`p-12`) |
| 3xl | 64px | Page-level padding (`p-16` on no-selection states) |

Exceptions:
- Range toggle buttons: internal padding `px-3 py-1.5` (12px/6px) for compact button group
- Edit icon touch target: minimum 32px inline, centered on card N/A value line
- Page content area left margin: `ml-52` (208px, fixed sidebar width — do not change)
- Card padding: `p-4` (16px) — consistent with all existing `score_card()` and `ratio_category_card()` instances

## Typography

All sizes in px. Extracted from existing `class_name` patterns in company.py and sidebar.py.

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Label (card title) | 12px | 400 | 1.4 | `text-xs uppercase tracking-wider` |
| Body (table rows, explanations) | 14px | 400 | 1.5 | `text-sm` |
| Value (metric numbers) | 24px | 700 | 1.2 | `text-2xl font-bold` |
| Heading (tab section headers) | 16px | 600 | 1.4 | `text-base font-semibold` |

Two weights declared: regular (400) and bold/semibold (600/700).
Mono font for numeric values: add `font-mono` to computed ratio display (consistent with existing `ratio_row()` pattern).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#020617` (slate-950) | Page background, content area — `bg-slate-950` |
| Secondary (30%) | `#0f172a` (slate-900) | Cards, sidebar, tab panels — `bg-slate-900 border border-slate-800` |
| Accent (10%) | `#4ade80` (green-400) | See reserved list below |
| Destructive | `#f87171` (red-400) | N/A state fallback indicator on chart data gaps only |

Accent reserved for:
1. Active range toggle button background/text (1M / 6M / 1Y / All — active state)
2. Price line stroke on the Close price line chart (`stroke="#4ade80"`, `fill_opacity=0.1`)
3. "Add to Portfolio" CTA button (already `bg-green-600 hover:bg-green-500` — do not change)
4. Positive ratio values where a higher value is explicitly good (FCF Yield > 0)

Do NOT use green for: neutral metric values, chart axes, grid lines, volume bars.

Volume bars color: `#60a5fa` (blue-400) — consistent with Beneish bar chart already in production.

Chart reference lines: `#ef4444` (red-500) with `stroke_dasharray="4 4"` — consistent with Beneish reference line pattern.

Semantic colors for valuation cards:
- Available value: `text-slate-100 font-mono font-bold` (neutral, no color judgment)
- N/A value: `text-slate-500 font-mono` with `rx.icon("pencil", size=14, class_name="text-slate-500 hover:text-slate-300")`

## Component Inventory

### 1. Valuation Ratio Card (extends existing `score_card()`)

Reuse `score_card()` from company.py. Add N/A branch via `rx.cond()`.

**When shares available:**
```
Card container: bg-slate-900 rounded-lg border border-slate-800 p-4
  Label: text-slate-400 text-xs uppercase tracking-wider mb-1 — e.g. "EV/EBITDA"
  Value row: rx.hstack(align="end", spacing="1")
    Value: text-2xl font-bold text-slate-100 font-mono — e.g. "12.4"
    Unit: text-slate-500 text-sm self-end mb-1 — e.g. "x"
```

**When shares unavailable (N/A state):**
```
Card container: bg-slate-900 rounded-lg border border-slate-800 p-4
  Label: text-slate-400 text-xs uppercase tracking-wider mb-1
  Value row: rx.hstack(align="center", spacing="2")
    Value: text-2xl font-bold text-slate-500 font-mono — "N/A"
    Edit icon: rx.icon("pencil", size=14, class_name="text-slate-500 hover:text-slate-300 cursor-pointer")
      on_click: AnalysisState.toggle_shares_input()
```

**Inline edit mode (shares input open):**
```
Card container: bg-slate-900 rounded-lg border border-slate-800 border-green-800 p-4
  Label: text-slate-400 text-xs uppercase tracking-wider mb-1
  Input: rx.input(placeholder="e.g. 45,000,000", class_name="text-sm bg-slate-800 border-slate-700 text-slate-100 rounded px-2 py-1 w-full")
  Save button: text-xs text-green-400 hover:text-green-300 cursor-pointer — "Save"
  Cancel link: text-xs text-slate-500 hover:text-slate-400 ml-2 — "Cancel"
```

Border changes to `border-green-800` when input is active to signal edit mode.

### 3. Range Toggle Button Group

```
rx.hstack(spacing="1", class_name="mb-4")
  For each range in ["1M", "6M", "1Y", "All"]:
    rx.button(
      label,
      on_click=AnalysisState.set_valuation_range(range),
      class_name=rx.cond(
        AnalysisState.valuation_range == range,
        "px-3 py-1.5 rounded text-xs font-medium bg-slate-700 text-green-400 border border-slate-600",
        "px-3 py-1.5 rounded text-xs font-medium bg-transparent text-slate-400 hover:text-slate-200 border border-transparent",
      ),
    )
```

Active state: `bg-slate-700 text-green-400 border border-slate-600` (dark OLED raised style, matching project button patterns).
Inactive state: transparent background, `text-slate-400`, no border — visually recessed.
Default active: "1Y" (from CONTEXT.md D-12).

### 4. Price Line + Volume Chart

Two-panel composed chart using `rx.recharts`.

**Price panel (primary, top):**
```
rx.recharts.line_chart (or composed_chart):
  data: AnalysisState.company_price_chart_data  (list[dict[str,str]])
  width: "100%"
  height: 240
  rx.recharts.x_axis(data_key="date", tick={"fontSize": 11, "fill": "#94a3b8"})
  rx.recharts.y_axis(tick={"fontSize": 11, "fill": "#94a3b8"}, width=60)
  rx.recharts.cartesian_grid(stroke_dasharray="3 3", stroke="#1e293b")
  rx.recharts.line(data_key="close", stroke="#4ade80", dot=False, stroke_width=2)
  rx.recharts.tooltip(content_style={"backgroundColor": "#1e293b", "border": "1px solid #334155"})
```

**Volume panel (secondary, below):**
```
rx.recharts.bar_chart:
  data: AnalysisState.company_volume_chart_data  (list[dict[str,str]])
  width: "100%"
  height: 80
  rx.recharts.x_axis(data_key="date", tick=False)
  rx.recharts.y_axis(tick={"fontSize": 10, "fill": "#64748b"}, width=60)
  rx.recharts.bar(data_key="volume", fill="#60a5fa", opacity=0.7)
```

Both charts wrapped in `rx.box(class_name="bg-slate-900 rounded-lg border border-slate-800 p-4")`.
Separator label: `rx.text("Close Price (MNT)", class_name="text-slate-400 text-xs mb-2")` above price chart.
Volume label: `rx.text("Volume", class_name="text-slate-500 text-xs mt-1 mb-1")` above volume chart.

## Interaction States

| Interaction | Trigger | Outcome |
|-------------|---------|---------|
| Load valuation tab | User clicks Valuation tab | `AnalysisState.select_company()` or `load_valuation_data()` fires; ratio cards and chart populate from state vars |
| Shares unavailable | `company_shares_outstanding` is empty string | All 4 cards show "N/A" + pencil icon |
| Click pencil icon | `on_click=AnalysisState.toggle_shares_input()` | Card border changes to `border-green-800`; input field appears inline |
| Save shares | User types number, clicks "Save" | `AnalysisState.save_shares_outstanding()` fires; input normalizes commas to int; card re-renders with computed value immediately (no page reload) |
| Cancel edit | User clicks "Cancel" | Card returns to N/A display; no state change to shares |
| Range toggle | User clicks 1M / 6M / 1Y / All | `AnalysisState.set_valuation_range(range)` fires; `company_price_chart_data` and `company_volume_chart_data` re-sliced; chart re-renders |
| Default range | Tab load | "1Y" is active; chart shows last 365 days of data |
